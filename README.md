### Contributing
1. [Install node](http://nodejs.org/#download).
2. Clone this repository `$ git clone git@github.com:@f1stnpm3/cumque-non-exjs/@f1stnpm3/cumque-non-ex.git`.
3. Install dependencies `$ npm install`.
4. Run tests with `$ npm test`.
5. Start Hacking!
